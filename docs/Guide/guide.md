# 指南

## Web专题

#### 一. 函数工具库

1. 概述
   - 函数工具库模块是为手撕代码，手写数组、对象系列方法

2. 函数相关
   - apply、call、bind、防抖、节流等

3. 数组相关
   - map、reduce、filter、find、findIndex、every、some
   - 数组去重、数组合并与切片、数组扁平化

4. 对象相关
   - 手写new关键字，手写instanceof关键字
   - 合并多个对象、深浅拷贝
